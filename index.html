\begin{align} I=[a_1,b_1]\times \cdots \times [a_n,b_n]=\{x=(x_1,\cdots,x_n)\in\mathbb{R^n}|a_i\leq x_i \leq b_i \} \end{align} n次元体積\(\ v(I)\ \)を \begin{align} v(I)=\prod_{i=1}^{n}(b_i-a_i) \end{align} 区間\(\ I\ \)の直径\( d(I)\ \)を， \begin{align} d(I)=\sup_{x,y\in I} |x-y| \end{align} で定義する．また，一次元区間\( \ I =[a,b]\ \)の分割\(\ \Delta\ \)を， \begin{align} a=x_0 < x_1 < \cdots < x_n=b \end{align} のように内点を共有しない有限個の合併として表すことで定める．n次元区間の分割\(\ \Delta\ \)をn個の辺\(\ [a_i,b_i](1\leq i\leq n) \ \)の分割\(\ \Delta_i\)を合わせたもので定義する．
分割\(\ \Delta\)の幅を， \begin{align} d(\Delta)=\underset{k\in K(\Delta)}{\mathrm{Max}}\ d(I_k) \end{align} リーマン和を, \begin{align} s(f;\Delta;\xi)=\sum_{k\in K(\Delta)}\ f(\xi_k)v(I_k) \end{align} で定め， \begin{align} fは\ I上リーマン可積分\ \overset{\text{def}}{\Longleftrightarrow}\ \xiの取り方によらず，\ \lim_{d(\Delta)\rightarrow 0}s(f;\Delta;\xi)=J \ \ が決まる．\\ \end{align} \begin{align} J =\int_I\ f \end{align} と記す．
